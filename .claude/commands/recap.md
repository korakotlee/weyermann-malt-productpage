# /recap - Fresh Start Context Summary

Quick catch-up for when you're starting a fresh session. Shows recent context, plans, commits, and learnings.

## Usage

```
/recap    # Get caught up on recent context
```

## What You Get
- Recent context issues (from ccc)
- Active plan issues
- Last 10 commits with timestamps
- Latest 3 retrospectives with focus
- Latest 3 learnings

## Action

Use the Task tool with:
```
subagent_type: context-finder
model: haiku
prompt: No arguments - run DEFAULT MODE for fresh start context summary.
```
